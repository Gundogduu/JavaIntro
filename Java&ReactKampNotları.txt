2.Ders: OOP(object oriented programing)

package oopIntro;

public class Product {
	
	//constructor
	public Product() {
		System.out.println("Ben çalıştım");
	}
	
	//overloading:aşırı yükleme
	//constructor'daki parametrelerin Product class'ındaki attribute'lerle bir bağlantısı yoktu.Ama "this" ile eşleştirdik.
	public Product(int id, String name, double unitPrice, String detail) {
		this();
		this.id = id;
		this.name = name;
		this.unitPrice = unitPrice;
		this.detail = detail;
	}
	
	int id;
	String name;
	double unitPrice;
	String detail;
	
	
}
Kural: Bir constructor çalışmak zorunda, birden fazla constructor varsa sadece biri çalışmak zorunda.
Eğer birden fazla constructor varsa ve birinde diğerinide çalıştırmak istersen; örneğin parametresiz constructor için "this();" yazman gerekiyor.
